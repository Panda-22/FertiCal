# FertiCal

一个用于计算理论灌溉液元素浓度的前端小工具。

页面支持上传水质检测 `Excel` 文件，配置 A/B 两桶母液中的肥料种类、用量、母液体积和稀释倍数，自动计算：

- 原水各元素背景值
- A/B 桶肥料带来的元素总量
- 理论灌溉液中的元素浓度
- 与目标值的偏差对比
- A / B 桶中 `CaSO4` 和 `Ca3(PO4)2` 的离子积与参考溶度积

## 功能概览

- 上传 `.xlsx`、`.xls`、`.csv` 水质检测报告
- 自动识别常见水质字段，如 `Ca`、`Mg`、`NO3-N`、`NH4-N`、`HCO3-`、`EC`、`pH`
- 分别配置 A 桶和 B 桶的肥料组合
- 支持 `kg` / `g` 两种输入单位
- 结果区以 `mmol/L` 展示主要元素浓度
- 支持录入元素目标值，并在偏差超过 `15%` 时红色提示

## 使用方法

### 方式一：直接打开

直接在浏览器中打开 [index.html](./index.html) 即可使用。

适合快速本地试用。

### 方式二：在本地静态服务中打开

如果希望更稳定地使用本地文件上传和浏览器调试，可以在项目目录启动一个静态文件服务，再通过浏览器访问对应地址。

## 页面操作说明

1. 在“导入水质检测报告”区域上传水质检测 `Excel` 文件。
2. 检查或手动修正识别出来的原水元素值。
3. 在 A 桶和 B 桶中分别添加肥料，填写用量、单位、母液体积和稀释倍数。
4. 在“元素目标值”区域填写目标浓度。
5. 在结果区查看：
   - A/B 配方元素总量
   - 理论进液浓度
   - 与目标值的偏差
   - A/B 桶析出风险

## 计算说明

### 1. 元素总量

根据肥料中各元素的质量百分比和输入用量，计算 A/B 桶中每种元素的总质量。

### 2. 理论灌溉液浓度

先根据母液体积计算母液中的元素浓度，再按稀释倍数换算为理论进液浓度，并叠加原水背景值。

主要元素结果以 `mmol/L` 展示，`EC` 和 `pH` 保留原单位。

### 3. 目标值偏差

当设置了目标值后，页面会计算：

```text
偏差 = (理论进液值 - 目标值) / 目标值
```

当偏差绝对值大于 `15%` 时，该元素的理论值和偏差会用红色显示。

### 4. 析出风险

页面会结合母液中的 `Ca`、`SO4`、`PO4` 以及原水中对应背景值，计算：

- `CaSO4`
- `Ca3(PO4)2`

并给出离子积与参考溶度积，便于判断母液中潜在析出风险。
