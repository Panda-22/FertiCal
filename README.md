# FertiCal

用于计算理论灌溉液元素浓度的网页工具，适合无土栽培、水肥配方核算和 A/B 母液方案复核。

![FertiCal 项目截图](./assets/fertical-home.svg)

## 能做什么

- 上传 `.xlsx`、`.xls`、`.csv` 水质检测报告
- 自动识别常见水质字段，如 `Ca`、`Mg`、`NO3-N`、`NH4-N`、`HCO3-`、`EC`、`pH`
- 分别配置 A 桶和 B 桶的肥料种类、用量、母液体积和稀释倍数
- 自动计算 A/B 配方中的元素总量
- 将理论进液浓度换算为 `mmol/L`
- 录入元素目标值，并在偏差超过 `15%` 时高亮提示
- 计算 A / B 桶中 `CaSO4` 和 `Ca3(PO4)2` 的离子积与参考溶度积

## 项目截图

当前页面包含四个主要区域：

1. 水质检测报告导入与原水数据修正
2. A 桶母液配置
3. B 桶母液配置
4. 理论灌溉液结果、目标值对比与析出风险分析

## 快速开始

### 直接打开

直接在浏览器中打开 [index.html](./index.html) 即可使用。

### 本地安装依赖

```bash
npm install
```

依赖主要用于浏览器端解析 Excel 文件。

## 使用方法

1. 在“导入水质检测报告”区域上传水质检测 `Excel` 文件。
2. 检查或手动修正识别出来的原水元素值。
3. 在 A 桶和 B 桶中分别添加肥料，填写用量、单位、母液体积和稀释倍数。
4. 在“元素目标值”区域填写目标浓度。
5. 在结果区查看：
   - A/B 配方元素总量
   - 理论进液浓度
   - 与目标值的偏差
   - A/B 桶析出风险

## 计算逻辑

### 元素总量

根据肥料中各元素的质量百分比和输入用量，计算 A/B 桶中每种元素的总质量。

### 理论灌溉液浓度

先根据母液体积计算母液中的元素浓度，再按稀释倍数换算为理论进液浓度，并叠加原水背景值。

主要元素结果以 `mmol/L` 展示，`EC` 和 `pH` 保留原单位。

### 目标值偏差

当设置了目标值后，页面会计算：

```text
偏差 = (理论进液值 - 目标值) / 目标值
```

当偏差绝对值大于 `15%` 时，该元素的理论值和偏差会用红色显示。

### 析出风险

页面会结合母液中的 `Ca`、`SO4`、`PO4` 以及原水中对应背景值，计算：

- `CaSO4`
- `Ca3(PO4)2`

并给出离子积与参考溶度积，便于判断母液中潜在析出风险。
